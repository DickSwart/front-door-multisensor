## Parts Used
- Power Jack (5.5mm barrel)
- Buck Converters‎ (LM2596)
- DHT22 module
- PIR sensor (HC-SR501)
- N-Channel MOSFET (IRF530N)
- Photoresistor
- 10kΩ Resistor
- Reed switch
- Siren (Large Tweety Pie - 119dB)
- NodeMCU V3.0
